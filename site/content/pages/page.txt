# Welcome to Station0

A lightweight flat-file CMS. Pages live as plain `.txt` files — no database migrations, fully Git-friendly.

## Features

- **Markdown** content with front matter
- **Block-based** page composition via YAML
- **Nested pages** with clean URLs (`/about/team`)
- **Admin panel** with page editor and user management
- Twig templates with full layout inheritance
